ESPUTNIK VELOCITY RULES:
- Always use $! (silent reference): $!data.get('field'), $!item.get('field')
- Items loop: #foreach($item in $data.get('items')) ... #end
- Conditionals: #if($!data.get('field')) ... #end
- Recommendations: #foreach($rec in $data.get('recommendedItems')) ... #end
- NEVER use {{ }}, {% %}, or bare $data.get() without !

VELOCITY DATA VARIABLES available in all templates:
- $!data.get('externalOrderId') — order number e.g. "#1001"
- $!data.get('date') — order date
- $!data.get('firstName'), $!data.get('lastName')
- $!data.get('email'), $!data.get('phone')
