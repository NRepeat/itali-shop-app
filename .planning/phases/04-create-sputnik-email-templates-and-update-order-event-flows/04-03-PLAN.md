ESPUTNIK VELOCITY RULES:
- Always use $! silent references: $!data.get('field'), $!item.get('field'), $!rec.get('field')
- Items loop: #foreach($item in $data.get('items')) ... #end
- Recommendations loop: #foreach($rec in $data.get('recommendedItems')) ... #end
- All optional fields: #if($!data.get('field')) ... #end
- NEVER use {{ }}, {% %}, or bare $data.get() without !
